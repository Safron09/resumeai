## What This Is
A web app that stores a user's professional profile once, then generates 
tailored, ATS-optimized, humanized resumes for any job description using AI

## The Problem It Solves
Job seekers waste hours rewriting resumes for every application. Existing tools make you start from scratch each time. ATS systems reject great candidates. AI-written resumes get flagged as robotic.

## Onboarding — done once
Upload LinkedIn PDF: Export your profile from LinkedIn — no scraping, no API, you own your data
Upload existing resume: PDF or DOCX — parsed and stored in your profile
Profile saved forever: Never re-enter your information again
